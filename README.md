# Menoh-Java
Java binding for [Menoh](https://github.com/pfnet-research/menoh/) DNN inference library.

This library is target for 64-bit architecture at the moment.

# Requirements
- [Menoh](https://github.com/pfnet-research/menoh/)

# Build
```bash
$ mvn package
```
